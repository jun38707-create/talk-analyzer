# 대화 맥락 분석기 사용 설명서

이 프로그램은 핸드폰 녹음 파일이나 대화 내용을 AI(Gemini 1.5 Flash)로 분석하여 '나'와 '상대방'의 주장을 요약해주는 로컬 웹 애플리케이션입니다.

## 📁 구성 파일
- `index.html`: 프로그램의 메인 화면 (실행 파일)
- `styles.css`: 디자인 및 레이아웃 설정 파일
- `app.js`: 분석 로직 및 Gemini API 연동 파일

## 🚀 시작하기
1. **Google AI Studio API 키 발급**:
   - [Google AI Studio](https://aistudio.google.com/app/apikey)에 접속하여 무료 API 키를 발급받으세요.
2. **프로그램 실행**:
   - `index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)로 엽니다.
3. **API 키 설정**:
   - 화면 상단의 입력창에 발급받은 API 키를 넣고 '저장' 버튼을 누릅니다. (키는 브라우저에 안전하게 저장됩니다.)
4. **파일 분석**:
   - 분석하고 싶은 녹음 파일(mp3, m4a 등)이나 텍스트 파일(txt)을 드래그 앤 드롭으로 업로드합니다.
   - '분석 시작하기' 버튼을 누르고 잠시 기다리면 리포트가 생성됩니다.

## 📱 모바일 사용 안내
- 이 폴더를 서버에 올리거나(GitHub Pages 등) 로컬 IP를 통해 모바일 브라우저에서도 동일하게 사용할 수 있습니다.
- 반응형 디자인이 적용되어 있어 스마트폰에서도 편리하게 확인 가능합니다.

---
**주의사항**: 오디오 분석의 경우 파일 용량이 너무 크면 분석 시간이 길어질 수 있습니다.
